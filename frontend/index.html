About Us

PureCare: Enterprise Application Integration Project

PureCare is a web-based healthcare management system developed as part of an Enterprise Application Integration project by a team consisting of Zahra Denisa Salsabila, Kayla Alifa Nathania, Siti Nurhasanah, Rayhan Putra Ferlian, dan Affan Maulana from the Information Systems program at Telkom University (Class S14802). This system provides features for managing patients, doctors, consultations, and medical records within a single integrated platform, demonstrating API integration concepts where each module functions like a separate service (CRUD) and can be tested using tools such as Postman to ensure proper data flow.
