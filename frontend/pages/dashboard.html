Selamat Datang di Smart Healthcare System

Sistem manajemen kesehatan berbasis microservices dengan arsitektur terdistribusi.


Fitur Utama:

  • Manajemen Pasien: Daftar, kelola, dan pantau data pasien
  • Manajemen Dokter: Kelola data dokter dan spesialisasi mereka
  • Konsultasi: Catat riwayat konsultasi pasien dengan dokter
  • Rekam Medis: Kelola rekam medis dan diagnosis pasien

Teknologi:

  • Backend: Node.js + Express (Microservices)
  • Database: MySQL
  • API Gateway: Express dengan Proxy Middleware
  • Frontend: HTML5, CSS3, JavaScript Vanilla
